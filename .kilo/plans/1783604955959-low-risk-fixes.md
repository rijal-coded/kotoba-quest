## 3. Replace non-null assertions with safe access (itemGenerator.ts)

**File:** `src/utils/itemGenerator.ts`  
**Lines 36–37:**

```
// Before
if (useHistorical && historical!.length > 0) {
  base = pickRandom(historical!);

// After
if (useHistorical && historical.length > 0) {
  base = pickRandom(historical);
```

The `if` already checks `historical.length > 0`, so `historical!` is redundant. TypeScript can narrow `historical` after the guard because `useHistorical` was already assigned based on the same `historical.length > 0` check. No runtime change.
